# StackOverflowApp (WIP)

A Stack Overflow-inspired Android app built with Kotlin and Jetpack Compose.

## Overview
This project is an Android app scaffold focused on modern Android development practices and a clean project structure. It is currently in the early stages and will be expanded with data, repositories, and user-facing features.

## Tech Stack
- Kotlin
- Jetpack Compose
- Material 3
- AndroidX ViewModel (planned/in progress)
- Repository pattern (planned/in progress)

## Current Status
- Initial project setup
- Compose theme integration
- Loading UI placeholder
- Basic package structure and app architecture setup

## Goals
- Clean, maintainable architecture
- Reusable composable UI components
- Scalable feature structure
- Good performance and stability foundations

## Run Locally
1. Open the project in Android Studio
2. Sync Gradle
3. Run the `app` module on an emulator or Android device
